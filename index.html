V8 - 2D Game Engine

Project link

V8 Game Engine

V8 is an open source game engine developed for the Love2D framework. V8 combines physics, lighting, dynamic objects, particles, and much more into an intuitive and easy to use system.

Since V8 is open source, it is constantly being improved and updated. You can contribute to the development of V8 here!


PowerScout

Project link

PowerScout

PowerScout is a web-app for tracking progress in the FIRST Robotics competitions.

This program is built for teams who are looking to improve their scouting performance, get real time analytical data, or just get an overview of the competition.

ScriptShare

Project link

ScriptShare

ScriptShare is a web-app for hosting code. While many websites like GitHub or Bitbucket integrate with Git, ScriptShare (while Git is supported) has file management built right into your project page.

ScriptShare also offers a help section where people can ask about anything they're having trouble with. User feedback is archived for access at any time.
